## Development Guidelines

### When Starting a New Session
1. Read this file (claude-progress.txt) first
2. Review recent git commits: `git log --oneline -10`
3. Check features.json for current feature status
4. Run `npm run dev` to verify app state
5. Choose ONE feature from features.json to work on

### During Development
1. Work on ONE feature at a time
2. Test feature thoroughly before marking complete
3. Update features.json status field
4. Commit with descriptive message: `feat: [feature-id] description`
5. Keep this file updated with progress

### Session End Checklist
1. Update this file with:
   - Features completed
   - Current status
   - Blockers encountered
   - Next steps
2. Update features.json with feature statuses
3. Commit progress: `git commit -m "chore: update progress after session"`
4. Ensure code is in working state
